The Process

We develop applications using a design led, agile approach. We do this to make sure everything we build is user-centred, and therefor has the best chance of being used and having impact in the area of engagement.

We will follow this process irrespective of the size of the project, although the degree to which we implement each step might vary.

Project Scoping phase

  • New Project Request
  • Project Scoping
  • The Estimate
  • Go / No Go decision

Project Development Phase

  • Team formation / kickoff meeting
  • User Research
  • Design Sprint(s)
  • Development Sprints
  • Deploy Prototype
  • Test and Learn
  • Launch
